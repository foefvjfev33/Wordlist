# 🔥 Wordlist Generator Pro - صانع قوائم كلمات مرور ذكي 🔥

برنامج واجهته رسومية GUI مبني بلغة Python باستخدام Tkinter لتوليد **قوائم كلمات مرور احترافية** بناءً على بيانات الضحية أو بشكل عشوائي. يدعم أنماط توليد ذكية، استبدال الأحرف، الرموز، الحفظ التلقائي، والمعاينة قبل الحفظ.

---

## 🧠 المميزات

- ✅ توليد كلمات مرور مبنية على بيانات شخصية (الاسم، المدينة، سنة الميلاد...)
- 🔄 دعم استبدال الحروف الذكي (مثل `a → @`, `e → 3`, ...)
- 🔢 توليد كلمات مرور عشوائية قابلة للتخصيص (أحرف، أرقام، رموز)
- 🧩 أنماط توليد مخصصة مثل: `{}` + `{}` أو `{}` + `_` + `{}`
- 🔐 دعم الحد الأدنى والأقصى لطول الكلمات
- 💾 إمكانية حفظ النتائج في ملف `.txt`
- 👀 معاينة النتائج قبل الحفظ
- 🌐 واجهة رسومية سهلة الاستخدام تدعم اللغة العربية


## 🚀 طريقة التشغيل

1. تأكد من وجود Python 3 مثبت على جهازك.
2. شغل البرنامج مباشرة:

```bash
python wordlist_generator.py
```
